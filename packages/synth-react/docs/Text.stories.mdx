# Text

The `<Text>` component is a layout primitive used to display a container that supports CSS text-related properties.

<Preview withToolbar>
  <Story name="intro">
    <div>
      <Text>
        Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod
        tempor incididunt ut labore et dolore magna aliqua.
      </Text>
      <br />
      <Text color="meta">
        Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod
        tempor incididunt ut labore et dolore magna aliqua.
      </Text>
    </div>
  </Story>
</Preview>

You can use the `<Text>` primitive without the default values from your tokens by passing the `withoutDefaults` prop.

<Preview withToolbar>
  <Story name="withoutDefaults">
    <Text withoutDefaults>
      Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod
      tempor incididunt ut labore et dolore magna aliqua.
    </Text>
  </Story>
</Preview>
