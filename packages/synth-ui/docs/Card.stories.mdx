# Card

The `<Card>` component is the base for all components that are used to highlighting content and distinguish it from other elements on the screen.

## Props

| Name      | Description                                                                      | Type                           | Default |
| :-------- | :------------------------------------------------------------------------------- | :----------------------------- | :------ |
| className | Required to properly extend styled-components.                                   | `string`                       | `''`    |
| children  | The elements you want to display within the card. | `React.ReactNode`           | `null`  |
| ref       | A React `ref` object that will point to the card's DOM element.                | `React.Ref<HTMLDivElement>` | `null`  |
