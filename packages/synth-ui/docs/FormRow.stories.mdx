# FormRow

The `<FormRow>` component is used to group form elements under one label.

<Preview>
  <Story name="default">
    <CardWrapper>
      <Card>
        <FormRow label="Email">
          <Input placeholder="email@example.com" />
        </FormRow>
        <FormRow>
          <Button>Submit</Button>
        </FormRow>
      </Card>
    </CardWrapper>
  </Story>
</Preview>

## Props

| Name      | Description                                                                                       | Type                        | Default |
| :-------- | :------------------------------------------------------------------------------------------------ | :-------------------------- | :------ |
| centered  | Boolean flag to determine if the elements in the row should be centered.                          | `boolean`                   | `false` |
| children  | The elements you want to display within this component.                                           | `React.ReactNode`           | `null`  |
| clasName  | Required to properly extend styled-components.                                                    | `string`                    | `''`    |
| fullWidth | Boolean flag to determine if the each element(s) in the row should take all the horizontal space. | `boolean`                   | `false` |
| inline    | Boolean flag to determine if the each element(s) in the row should be displayed inline.           | `boolean`                   | `false` |
| ref       | A React `ref` object that will point to the component's DOM element.                              | `React.Ref<HTMLDivElement>` | `null`  |

## Positioning Form Elements

Form elements whitin a `<FormRow>` could be displayed inline by using the `inline` prop.

<Preview>
  <Story name="inline">
    <CardWrapper>
      <Card>
        <FormRow inline>
          <Input />
          <Button>Button</Button>
        </FormRow>
        <FormRow inline>
          <Button>Button A</Button>
          <Button>Button B</Button>
          <Button>Button C</Button>
        </FormRow>
      </Card>
    </CardWrapper>
  </Story>
</Preview>

Form elements whitin a `<FormRow>` could be displayed at full width, i.e., taking all the horizontal space available.
