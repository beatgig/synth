# Editable

The `<Editable>` component is variant of the `<Input>` component that displays a text input without background and border colors.

## Props

| Name      | Description                                                      | Type                          | Default |
| :-------- | :--------------------------------------------------------------- | :---------------------------- | :------ |
| className | Required to properly extend styled-components.                   | `string`                      | `''`    |
| ref       | A React `ref` object that will point to the input's DOM element. | `React.Ref<HTMLInputElement>` | `null`  |
