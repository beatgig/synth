# Clickable

The `<Clickable>` component is a variant of the `<Button>` component that displays a clickable element with no background and border colors.

<Preview withToolbar>
  <Story name="default">
    <Clickable>Clickable</Clickable>
  </Story>
</Preview>

## Props

| Name  | Description                                   | Type     | Default |
| :---- | :-------------------------------------------- | :------- | :------ |
| children | The elements you want to display within the button. | `React.ReactNode` | `null` |
| className | Required to properly extend styled-components.| `string` | `''` |  
| icon | An optional icon component that can be rendered along side the button's content. | `React.ReactElement` | `null`|
| iconPosition | The icon's position. | `'top'` \| `'right'` \| `'bottom'` \| `'left'` | `'left'` |
|intent| The component's intent will determine it's text and background color. | `'none'` \| `'danger'` \| `'success'` \| `'highlight'` | `none` |
| ref | A React `ref` object that will point to the button's DOM element. | `React.Ref<HTMLButtonElement>` | `null`|

## Intent

The `intent` property provides button with more context related to the action(s) they can trigger.

## States

Use the `disabled` property to conditionally disable the button if necessary.

<Preview withToolbar>
  <Story name="disabled">
    <Clickable disabled>Disabled Clickable</Clickable>
  </Story>
</Preview>

## Displaying Icons

Clickables can contain a combination of a label and an icon; The `icon` property gives you the ability to pass in an icon component to be rendered with the button's text. The `iconPosition` property will let you specifiy the positioning of the icon.
