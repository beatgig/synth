# Flex

The `<Flex>` component is a generic container that uses `flex` as it's CSS `display` property. Use this component to layout other components.

<Preview>
  <Story name="intro">
    <Flex style={{ backgroundColor: 'rgba(0,0,0,0.25)', height: 100, padding: 15, width: '100%' }}>
      Content
    </Flex>
  </Story>
</Preview>

## Props

| Name      | Description                                                      | Type                          | Default |
| :-------- | :--------------------------------------------------------------- | :---------------------------- | :------ |
| children | The elements you want to display within the flex component. | `React.ReactNode` | `null` |
| className | Required to properly extend styled-components.                   | `string`                      | `''`    |
| row | A boolean flag to determine what the value of the `flex-direction` CSS property will be, if `true`, then the `flex-direction` CSS property will be `row`. | `boolean` | `true` |
| full | A boolean flag that if `true`, will set the element's `height` and `width` CSS properties to `100%`. | `boolean` | `false` |
| flex | The value to be assigned to the `flex` CSS property. | `number` \| `string` | `''` |
| grow | The value to be assigned to the `flex-grow` CSS property. | `number` \| `string` | `''` |
| wrap | A boolean flag to determine what the value of the `flex-wrap` CSS property will be, if `true`, then the `flex-wrap` CSS property will be `wrap`. | `boolean` | `false` |
| basis | The value to be assigned to the `flex-basis` CSS property. | `number` \| `string` | `''` |
| shrink | The value to be assigned to the `flex-shrink` CSS property. | `number` \| `string` | `''` |

| placeContent | The value to be assigned to the `place-content` CSS property. | `'center start'` \| `'start center'` \| `'end left'` \| `'flex-start center'` \| `'flex-end center'` \| `'baseline center'` \| `'first baseline space-evenly'` \| `'last baseline right'` \| `'space-between space-evenly'` \| `'space-around space-evenly'` \| `'space-evenly stretch'` \| `'stretch space-evenly'` \| `'inherit'` \| `'initial'` \| `'unset'` | `''` |

## Flexbox Properties

The `<Flex>` component has flexbox-specific properties that match their CSS counterparts, e.g., `alignItems`, `justifyContent`, `grow`, `wrap`, etc.
