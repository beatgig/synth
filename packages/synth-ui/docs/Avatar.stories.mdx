# Avatar

The `<Avatar>` component is used to render user avatars.

<Preview withToolbar>
  <Story name="intro">
    <Avatar src="https://randomuser.me/api/portraits/men/1.jpg" scale={5} />
  </Story>
</Preview>

## Props

| Name  | Description                                   | Type     | Default |
| :---- | :-------------------------------------------- | :------- | :------ |
| src\* | The URL of the embeddable content.            | `string` | `null`  |
| scale | A scale index from the `avatars` synth token. | `number` | `0`     |

## Avatar Sizes

Avatars can be rendered in different sizes which are taken from the `@avatars` global token.
