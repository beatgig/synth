# Button

Use buttons to give users the ability to trigger actions.

<Preview withToolbar>
  <Story name="default">
    <Button>Button</Button>
  </Story>
</Preview>

## Props

| Name  | Description                                   | Type     | Default |
| :---- | :-------------------------------------------- | :------- | :------ |
| children | The elements you want to display within the button. | `React.ReactNode` | `null` |
| className | Required to properly extend styled-components.| `string` | `''` |  
| icon | An optional icon component that can be rendered along side the button's content. | `React.ReactElement` | `null`|
| iconPosition | The icon's position. | `'top'` \| `'right'` \| `'bottom'` \| `'left'` | `'left'` |
|intent| The component's intent will determine it's text and background color. | `'none'` \| `'danger'` \| `'success'` \| `'highlight'` | `none` |
| outline| Displays the button only using it's outline. | `boolean` | `false`|
| ref | A React `ref` object that will point to the button's DOM element. | `React.Ref<HTMLButtonElement>` | `null`|

## Intent

The `intent` property provides button with more context related to the action(s) they can trigger.

## Secondary Actions

Using the `outline` property will give you the option to render a variant of the button that will only show the outline of the button. Use this button for secondary actions.

## States

Use the `disabled` property to conditionally disable the button if necessary.

## Displaying Icons

Buttons can contain a combination of a label and an icon; The `icon` property gives you the ability to pass in an icon component to be rendered with the button's text. The `iconPosition` property will let you specifiy the positioning of the icon.

<Preview withToolbar>
  <Story name="with icon">

      <Spacer right scale={2} />

        Search Button
      </Button>
      <Spacer right scale={2} />
      <Button icon={<SearchIcon />} iconPosition="top">
        Search Button
      </Button>      
      <Spacer right scale={2} />
      <Button icon={<SearchIcon />} iconPosition="bottom">
