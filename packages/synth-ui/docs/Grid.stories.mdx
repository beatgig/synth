## Props

| Name      | Description                                                                      | Type                           | Default |
| :-------- | :------------------------------------------------------------------------------- | :----------------------------- | :------ |
| children  | The elements you want to display within this component.                          | `React.ReactNode`              | `null`  |
| className | Required to properly extend styled-components.                                   | `string`                       | `''`    |
| columns   | The number of columns you want to display                                        | `number`                       | `0`     |
| icon      | An optional icon component that can be rendered along side the button's content. | `React.ReactElement`           | `null`  |
| ref       | A React `ref` object that will point to the button's DOM element.                | `React.Ref<HTMLButtonElement>` | `null`  |

## Grid Item Wrapping

You can control how `<GridItem>` wrap around by changing the `columns` property on their parent `<Grid>` component.
