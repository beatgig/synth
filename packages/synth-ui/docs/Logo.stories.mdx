# Logo

Use the `<FullLogo>` and `<MiniLogo>` components to display the BeatGig logo.

  <Story name="intro">

  </Story>

## Props

| Name      | Description                                                                      | Type                           | Default |
| :-------- | :------------------------------------------------------------------------------- | :----------------------------- | :------ |
| className | Required to properly extend styled-components.                                   | `string`                       | `''`    |
| dark      | A boolean flag to determine the color variant of the logo; The default is white. | `boolean`                      | `false` |
| ref       | A React `ref` object that will point to the icon's DOM element.                  | `React.Ref<HTMLButtonElement>` | `null`  |
