# Icon

The `<Icon>` component is used to build different icon components.

<Preview>
  <Story name="intro">
    <SearchIcon scale={5} />
  </Story>
</Preview>

## Props

| Name      | Description                                                       | Type                           | Default |
| :-------- | :---------------------------------------------------------------- | :----------------------------- | :------ |
| className | Required to properly extend styled-components.                    | `string`                       | `''`    |
| children  | The icon's markup.                                                | `React.ReactNode`              | `null`  |
| ref       | A React `ref` object that will point to the button's DOM element. | `React.Ref<HTMLButtonElement>` | `null`  |
| scale     | A scale index from the `icons` synth token.                       | `number`                       | `0`     |

## Icons

We currently use the open sourced icons from [Google Material Design](https://material.io/resources/icons).
