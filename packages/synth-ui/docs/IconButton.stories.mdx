# IconButton

The `<IconButton>` is a variant of the `<Button>` component that you can use to display buttons that will only contain icons.

<Preview>
  <Story name="intro">
    <IconButton icon={<CheckIcon />} />
  </Story>
</Preview>

## Props

| Name      | Description                                                                      | Type                           | Default |
| :-------- | :------------------------------------------------------------------------------- | :----------------------------- | :------ |
| className | Required to properly extend styled-components.                                   | `string`                       | `''`    |
| icon      | An optional icon component that can be rendered along side the button's content. | `React.ReactElement`           | `null`  |
| ref       | A React `ref` object that will point to the button's DOM element.                | `React.Ref<HTMLButtonElement>` | `null`  |

## Icon Sizes

The icons within the button can be rendered in different sizes which are taken from the `@icons` global token.
